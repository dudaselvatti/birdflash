# Music Diary

Uma aplicação web estática desenvolvida para funcionar como um diário musical interativo. O projeto organiza conteúdos em componentes expansíveis e consome uma API de clima em tempo real para sincronização de dados meteorológicos.

## Recursos

- **Interface Interativa:** Transições de tela suaves e manipulação dinâmica do DOM.
- **Sistema de Diário (Accordion):** Layout limpo com tópicos colapsáveis.
- **Clima em Tempo Real:** Integração assíncrona com API meteorológica pública.
- **Media Players:** Suporte a embeds de áudio responsivos.
- **Responsividade:** Design adaptável para dispositivos móveis e desktops.

## Tecnologias Utilizadas

- **HTML5:** Marcação semântica.
- **CSS3:** Estilização, animações e layout (Grid/Flexbox).
- **JavaScript (Vanilla):** Lógica de interface e requisições assíncronas (Fetch API).
- **Open-Meteo API:** Dados meteorológicos.
